# Scripts

### start

Start the project in development mode with the .env file that is in the root

```bash
yarn start
```

### test

Run the unit tests hosted on **tests**

```bash
yarn test
```

### build

Compile the project with the .env file that is in the root, its output is in dist/app.js

```bash
yarn build
```
