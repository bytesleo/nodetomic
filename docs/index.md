# NODETOMIC

Minimalist boilerplate for **nodejs**, designed for vertical and horizontal scalability.

## Technologies

- Express
- Redis 6
- MongoDB 4
- Swagger 3
- Webpack 5
- Babel 7
- Socket 4
- Eslint
- Prettier
- Jest
